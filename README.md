# Georgia Tech CS4641 - Machine Learning Group 7 Project  

## YouTube Clickability Study: Analyzing the Impact of Titles and Thumbnails  

### Project Overview  
This repository contains the implementation and analysis for a machine learning project focused on predicting YouTube video engagement based on titles and thumbnails. Developed as part of the CS4641 Machine Learning course at Georgia Tech (Fall 2025).
